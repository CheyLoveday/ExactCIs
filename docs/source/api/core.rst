Core Module
===========

The core module provides the fundamental functionality for calculating confidence intervals.

   :members:

   :show-inheritance:

Mathematical Functions
--------------------

The core module includes various mathematical functions used in confidence interval calculations, such as:

- Probability mass function (PMF) calculations
- Cumulative distribution function (CDF) calculations
- Root finding algorithms
- Optimization functions

Validation Functions
------------------

The core module also includes validation functions to ensure input parameters are valid:

- Table validation (checking for valid 2Ã—2 contingency tables)
- Parameter validation (checking for valid alpha values, etc.)

Optimization Algorithms
---------------------

The core module implements optimization algorithms for finding confidence interval bounds:

- Root finding for p-value functions
- Bracket expansion for finding initial bounds
- Refinement algorithms for improving precision
